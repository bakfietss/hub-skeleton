# Example Solution

> Dit is een voorbeeld solution template

## Gebruik

1. Kopieer deze folder naar een nieuwe naam
2. Pas de `.planning/` files aan
3. Begin met development

## Structure

```
solution/
├── .planning/
│   ├── PROJECT.md
│   ├── STATE.md
│   ├── BACKLOG.md
│   └── ROADMAP.md
├── agents/
│   └── README.md
├── src/
├── CLAUDE.md
└── README.md
```

## Verwijderen

Als je geen voorbeeld nodig hebt, verwijder deze `_example-solution` folder.
